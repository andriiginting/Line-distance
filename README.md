# CompareTwoLines

### Project Description:
this project about comparing two lines and calculate distance between two points

Dev Env Setup:
```
intellij/java sdk/gradle 4.8.1
```

Build Instructions:
```
./gradlew build
```

Run Instructions:
```
./gradlew run
```

Test Instructions:
```
./gradlew clean test
```
